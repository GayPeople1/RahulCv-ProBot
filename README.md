### Install (termux)
Clone this project

```bash
> pkg install git
> git clone https://github.com/GayPeople1/RahulCv-ProBot
> cd Btelegram-bot
```

Install the dependencies:

```bash
> pkg install python
> python install.py
```

### Usage
run the telegram bot

```bash
> python start.py
```

### Install (windows)
install nodejs first

```bash
> npm install
> npm install axios
> npm install telegraf
> npm install nodemon
> nodemon bot.js
> atau
> node bot.js
```
